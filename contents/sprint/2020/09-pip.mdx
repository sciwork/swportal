# Scisprint 2020 September: pip/pypa/packaging

Help make the entire packaging/distribution toolchain, as usable and
robust as possible!

The Python Packaging Authority (PyPA) is a working group that maintains
a core set of software projects used in Python packaging. Some notable
projects maintained by PyPA include:

- Warehouse (web interface to [pypi.org](https://pypi.org/) )
- [pip](https://github.com/pypa/pip/)
- [setuptools](https://github.com/pypa/setuptools/)
- [virtualenv](https://github.com/pypa/virtualenv)
- [pipenv](https://github.com/pypa/pipenv)
